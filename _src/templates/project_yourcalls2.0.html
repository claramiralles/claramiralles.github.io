Technical summary:

Single Page Application built in React to record information about client calls. Data visualisation for analysis purposes.Shown in graphics done by integrating React Google Charts.Web layout with HTML5 & CSS3.

Usage:

Yours Calls is a web app to register the phone calls that receive the employees from Interacso, the company we are working for. It includes a call history, call statistics and slack notifications.

Features:

- Phone calls registry and call history availability.
- Saved information about the call.
- Automatic notifications with the registered call data through Slack.
- This last functionality works using a private OAuth Access Token that is provided by Slack.
