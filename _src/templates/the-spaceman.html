The Spaceman is a desktop version game in which the user can scroll from left to right of the board dodging asteroids and collecting gems and oxygen.

The purpose of the game is for the user to survive as long as possible by taking oxygen.

Technologies:

User can earn points as they go taking oxygen and through the gems, and will lose the game when it hits against an asteroid.
