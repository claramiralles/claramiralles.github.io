Sand App

Front and backend development

Back

Sand is a collaborative WebApp that allows the user to consult the status of the beaches in real time.

Users update the information of the beaches through the forms they'll send us when we detect by geolocation that the user is there, the rest of the information is obtained from the weather API.

By completing the forms, users get points that they can later exchange for discounts related to the beaches.

Technologies:

The Admin user. It has a dynamic button that allows you to take charge of the content management of the App, you can then create / edit / delete beaches and discounts.
